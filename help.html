Getting started, general Help and FAQ sheet for Lecture and Assignments

1. Staying up to date

The latest version of the slides and assignments is kept in this gitLab repository. You should have access to the FAU-gitLab with your IdM account. 

git clone https://gitlab.cs.fau.de/i9/appvis.git
Remember to keep your local repository up to date. 
git pull
There are different tutorials on how to use git. For example this or others.

You can us any git GUI as well. Examples are GitKraken, Sourcetree and others.

2. Edit Assignments

You can find the Assignments in the aforementioned repository in the following path. Where XY is the number of the assignment. 

./Assignments/a_XY/a_XY.html
./Assignments/a_XY/assignmentXY.js
./Assignments/a_XY/assignmentXY_solution.js
In order to finish an assignment, you can edit these provided files to match the task description. The description is given both in the code(.js) and on the site(.html). You can choose any programming environment that suits your needs. Free examples are Visual Studio Code, Atom and others

To view the task description as well as to view your programmed results, you can open the html-site of the according assignment in your favoured browser. Note however that, not every browser may support every feature needed. We recommend using Firefox or Chrome.

Local Server

Due to the CORS Policy, the browser may forbid the access of module-files when opening the .html from your file system. One solution to this problem is to use a local server.

Both Visual Studio Code and Atom offer a plugin called Live Server, which - additionally to serving - updates the page when any file is changed.

Another possibility is to use a python server inside the repository-root directory 

python -m SimpleHTTPServer
To which you can connect using your browser. 
http://localhost:5500
